Política de Privacidad RYSA

Los datos proporcionados en este formulario serán utilizados exclusivamente para:

  • Analizar tus necesidades de branding y community manager
  • Preparar una propuesta personalizada para tu proyecto
  • Contactarte para aclarar dudas o solicitar información adicional
  • Crear un perfil en nuestra base de datos RYSA si decides trabajar con nosotros
  • Gestionar el sistema de referidos si aplica

Tus datos serán almacenados en la base de datos multipropósito de RYSA y solo serán accesibles para nuestro equipo autorizado según los roles establecidos (Admin, Diseñador, Community Manager, etc.).

No compartiremos tu información con terceros sin tu consentimiento explícito.
