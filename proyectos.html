Producciones Audiovisuales

Sistema completo de gestión de reservas para excursiones de montaña, con panel de administración, calendario integrado y pasarela de pagos.

Problema

Gestión manual de reservas, pagos desorganizados y falta de visibilidad de disponibilidad en tiempo real.

Solución

Plataforma web con autogestión, calendario interactivo, integración con Mercado Pago y panel de administración.

Tecnologías

PHP MySQL Bootstrap 5 JavaScript API Mercado Pago

Resultados

Reducción del 80% en tiempo de gestión, aumento del 40% en reservas online y sistema autosustentable.

Estudio Juridico Bucci

Sitio web institucional y tienda online integrados para estudio jurídico, con gestión de contenido, blog legal y e-commerce de servicios jurídicos.

Problema

Solución

Desarrollo de sitio web corporativo con tienda online integrada, permitiendo mostrar servicios y vender consultas/asesorías legalmente.

Tecnologías

HTML5 PHP MySQL CSS3 JavaScript
